./x.sh

./x.sh java-gitlab

使用目錄掛載:
docker run -d -p 8090:8090 --name javademo -v /root/java-server/java-gitlab-1.0-SNAPSHOT.jar:/app.jar openjdk:8 java -jar app.jar
